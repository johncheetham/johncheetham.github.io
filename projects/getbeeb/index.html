GetBeeb

GetBeeb - An open source program to generate HTML pages containing links to BBC audio files


GetBeeb is an open source java program that generates html pages containing links to BBC audio files (listen again files). My main reason for doing this was for use in my PDA which can't use iPlayer and has a small screen which struggles to load the BBC website pages.

Download Git Master: .tar.gz    .zip

Download last release: getbeeb-0.1.4.zip    October 2008

View project on GitHub

Getbeeb works by reading the experimental XML feeds provided by the BBC.
