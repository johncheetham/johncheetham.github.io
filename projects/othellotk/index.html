OthelloTk - An open source Othello Game

Download last release: othelloTk-0.1.1.tar.gz    othelloTk-0.1.1.zip    September 2015

View project on GitHub

OthelloTk is a gui for the Edax othello engine written in Python3 / Tkinter.

Features:

  • Load / Save Games
  • Set Engine Time Limit
  • Browse backwards and forwards through the game
  • Resizable main window

To use OthelloTk you need to download the Edax engine from the Edax Website
